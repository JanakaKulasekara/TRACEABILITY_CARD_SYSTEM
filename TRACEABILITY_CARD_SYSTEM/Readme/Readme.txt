Requirements

1) Windows XP/7 (Tested and working)
2) .NET Framework 4.0 (dotNetFx40_Full_x86_x64.exe)
3) Zebra GT800 EPL Printer (Tested) and Drivers -  Whatever printer used to print labels, should be set as Default printer.



Traceability Label - Zebra 
1) 2.50" * 1.75" (h*w in inches)
2) Speed 2"-3"/s
3) Required darkness
